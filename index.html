HTML

  • The head element contains information about the webpage.
    • The meta element contains information that helps the webpage render appropriately.
    • The most common tool in meta is to set charset="UTF-8".
    • The title element is used to give the webpage a name in the tabs area of the browser.
      • A good title contributes to better SEO performance.
  • The body element represents the visible content shown to the user.
    • There are a variety of tools that can and should be used in the body of the html.
      • The ul and ol tags are used for unordered and ordered lists respectively while li is for list items.
      • The tags h1 through h6 are used for different sized headers.
      • The section tags are given classes and id's to be distinguishable in the CSS style file.
    • The footer section comes at the very bottom of the body and is used for copyrights, org info, etc.
    • The body also contributes to SEO performance.

CSS

  • CSS is used to stylize an HTML web page and uses various attributes to do so.
    • A margin indicates how much space we want around the outside of an element.
    • A padding indicates how much space we want around the content inside an element.
    • A border indicates, well, precisely that, a border, which can be given a box-shadow element to go with it.
  • A class can be used to apply code to every instance of that class in the HTML.
  • Colors can also be defined in many ways in CSS; by semantic references, hexadecimal code, hex#, etc.
  • There are three ways to style a webpage using CSS: inline, internal, and external.
  • Finally, when using a .css stylesheet, it must be defined in the .html file's header with the link tag.

Git

  • The most common commands that we use in git are listed here
    • git status: checks what branch we are currently on
    • git checkout -b [branch-name]: creates a new branch and switches to it
    • git add -A: adds all files to to the stage for the next commit.
    • git commit -m “[descriptive message]”: commits the staged content as a new snapshot
  • This git cheat sheet is a very useful resource to come back to.

JavaScript

  • A variable is a named container that allows us to store data in our code.
    • Variables can contain a variety of data types: numeric, string, boolean, etc.
    • Variables need to be defined bofore they are called, otherwise an error will occur.
  • Control flow is the order in which a computer executes code in a script.
  • Functions are used to reuse code that will be needed in multiple different locations.
  • Loops are used to repeatedly do a task all at the same time.
    • These are often used when handling data of arrays, which are similar to lists of values.
    • Loops normally need three statements: the initializer, the condition (when to quit), and the iterater.
  • Javascript is used for interactions between the user and the web page.
