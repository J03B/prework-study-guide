HTML

  • The head element contains information about the webpage.
    • The meta element contains information that helps the webpage render appropriately.
    • The most common tool in meta is to set charset="UTF-8".
    • The title element is used to give the webpage a name in the tabs area of the browser.
        • A good title contributes to better SEO performance.
  • The body element represents the visible content shown to the user.
    • There are a variety of tools that can and should be used in the body of the html.
      • The ul and ol tags are used for unordered and ordered lists respectively while li is for list items.
      • The tags h1 through h6 are used for different sized headers.
      • The section tags are given classes and id's to be distinguishable in the CSS style file.
    • The footer section comes at the very bottom of the body and is used for copyrights, org info, etc.
    • The body also contributes to SEO performance.

Git

  • The most common commands that we use in git are listed here
    • git status: checks what branch we are currently on
    • git checkout -b [branch-name]: creates a new branch and switches to it
    • git add -A: adds all files to to the stage for the next commit.
    • git commit -m “[descriptive message]”: commits the staged content as a new snapshot
